注意：

1. 导入项目需要配置Modules和Artifacts。这一步会在使用IDEA的导入导入项目的时候给予提示，按照提示进行便好。
   如果没有提示，关闭项目，重新打开/导入。

2. 导入项目之后，如果不能成功的生成war包，或者运行时报错，那么需要删除前面配置的Artifacts，然后按照下述链接重新配置Artifacts配置便可。
